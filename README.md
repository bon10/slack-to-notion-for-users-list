## What is this?

Slack のユーザー情報を Notion の DB に書き込むための Go 製ツールです。

## How to use

create .env on top directory.

```
SLACK_TOKEN=<xoxb-xxxx>
NOTION_TOKEN=<secret_xxx>
NOTION_DATABASE_ID=<Notion DATABASE ID>
```
